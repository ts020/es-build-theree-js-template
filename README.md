# EsBuild + Typescript + Three.js ProjectTemplate


## Installation

### Clone repo

``` bash
# clone the repo
git git@github.com:ts020/es-build-theree-js-template.git my-project

# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install
```

### Copy and Paste

Copy all your files to your project folder and then,

``` bash
# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install
```


## development
```bash
# run dev server
$ npm run dev

# build typescript
$ npm run dev:ts

# reload browser
```

## Build
```bash
$ npm run build
```
